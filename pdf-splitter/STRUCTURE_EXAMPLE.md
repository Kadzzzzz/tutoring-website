# Structure de sortie attendue

## Exemple pour Chimie PCSI Semaine 6

```
public/documents/exercices/chimie/Colles-PCSI-S6/
├── Programme.pdf          # Page 1 du PDF original
├── P1.pdf                 # Planche 1 (1 seul exercice)
├── P2.pdf                 # Planche 2 (1 seul exercice)
└── P3.pdf                 # Planche 3 (1 seul exercice)
```

## Exemple pour Physique MPSI Semaine 7 (avec plusieurs exercices)

```
public/documents/exercices/physique/Colles-MPSI-S7/
├── Programme.pdf          # Page 1
├── P1-1.pdf               # Planche 1, Exercice 1
├── P1-2.pdf               # Planche 1, Exercice 2
├── P2.pdf                 # Planche 2 (1 seul exercice)
├── P3-1.pdf               # Planche 3, Exercice 1
├── P3-2.pdf               # Planche 3, Exercice 2
└── P4-1.pdf               # Planche 4 (bonus), Exercice 1
```

## Règles de nommage

- **Programme** : Toujours `Programme.pdf`
- **1 exercice par planche** : `P{n}.pdf` où n = numéro de planche
- **Plusieurs exercices** : `P{n}-{m}.pdf` où n = numéro planche, m = numéro exercice
